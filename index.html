Tag <h1> até <h6>

As tags <h1> a <h6> são usadas para definir cabeçalhos HTML. <h1> define o título mais importante. <h6> define o título menos importante.
Nota: Use apenas um <h1> por página - isso deve representar o título principal

Tag <p>

A tag <p> define um parágrafo.
Os navegadores adicionam automaticamente uma única linha em branco antes e depois de cada elemento <p>.

Tag <mark>

A tag <mark> define o texto que deve ser marcado ou destacado.

Tag <i>

A tag <i> deixa a palavra ou a frase em itálico.

Tag <u>

A tag <u> deixa a palavra ou a frase em sublinhado.

Tag <strong>

A tag <strong> deixa a palavra ou a frase em negrito.

Tag <ol>

A tag <ol> define uma lista ordenada. Uma lista ordenada pode ser numérica ou alfabética.

Tag <ul>

A tag <ul> define uma lista não ordenada (com marcadores).

Tag <li>

A tag <li> define item dentro da lista, ele é usando dentro de um ul ou ol.

Tag <a>

A tag <a> define um hiperlink, que é usado para vincular de uma página a outra.

Tag <hr>

A tag <hr> traça uma linha horizontal na pagina.

Tag <sub>

A tag <sub> define o texto subscrito. O texto subscrito aparece meio caractere abaixo do linha normal e às vezes é renderizada em uma fonte menor. O texto subscrito pode ser usado para fórmulas químicas, como H2O.

Tag <sup>

A tag <sup> define texto sobrescrito. O texto sobrescrito aparece meio caractere acima do linha normal e às vezes é renderizada em uma fonte menor. Texto sobrescrito pode ser usado para notas de rodapé, como WWW[1].

Tag <blockquote>

A tag <blockquote> especifica uma seção que é citada de outra fonte.

Tag <font>

A tag <font> foi usada no HTML 4 para especificar o tipo de fonte, o tamanho da fonte e a cor do texto.

Tag <del>

A tag <del> define o texto que foi excluído de um documento. Os navegadores geralmente marcam uma linha no texto excluído.

Tag <abbr>

A tag <abbr> define uma abreviatura ou um acrônimo, como "HTML", "CSS", "Sr.", "Dr.", "ASAP", "ATM".
